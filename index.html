Introduction to Cloud Computing

Cloud computing is the on-demand delivery of computing power, database storage, applications, and other IT resources through a cloud services platform via the internet with pay-as-you-go pricing.

Benefits

  • No Capital Expenditure
  • Pay as you go and Pay only for what you use
  • True Elastic Capacity.

The service services provided by a Cloud Provider can be majrly divided into three as given below.

  1. IAAS - Infrastructure as a Service
  2. PAAS - Platform as a Service
  3. Software as a Service

Cloud Provider

A cloud provider is a company that offers component of cloud computing as a service to other businesses or individuals. Cloud providers are sometimes referred to as cloud service providers or CSPs. Given below are some of the cloud providers who are playing a key role.

  • Amazon Web Services(AWS) by Amazon
  • Azure by Microsoft
  • Google Cloud Computing(GCS) by Google
